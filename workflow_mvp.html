End-to-End Booking Lifecycle
Covers Register → Login → Choose Service → View Vehicle → Submit → Payment → Pending → Confirmed → Dispatched → Completed.
Staff-Side Control
Control Staff can review, approve/reject, dispatch, and create phone bookings — maintaining full operational oversight.
Automated State Transitions
System manages PENDING → CONFIRMED → DISPATCHED → ON_ROUTE → ON_TRIP → COMPLETED automatically on each action.
Driver Lifecycle
Driver advances trip: On Route → On Trip → Completed. Notifications keep the customer informed at every stage.
Payment Flexibility
Supports Cash (pay on pickup) and PayPal (online). Handles payment failure with retry before queuing.
Cancel & Enquiry Handling
Customers can cancel before trip starts. Staff respond to enquiries within the same lifecycle.

Features excluded from MVP (Should/Could/Won't Have): Reviews, Loyalty Points, Face ID, Promotions, Membership, Community, GPS Tracking, Admin Dashboard.
